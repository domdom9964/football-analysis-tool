# football-analysis-tool
 A tool to build a football squad with stats and review the stats of every individual player and the entire squad to help team tactics and evaluate individual player performance


 ## features

 - Squad anschauen
 - Squad stats abrufen
 - Spieler hinzufügen
 - Spieler nach position Filtern
 - Ein aktives Main Menu
 - Spieler löschen
 - Spieler stats vergleichen


## Technologien

- Python

## Geplante Erweiterung

- Spieler bearbeiten
- Spielerbewertungssystem
- Information in JSON von Transfermarkt.de bekommen
- Spieler Stärken/Schwächen evaluiren
